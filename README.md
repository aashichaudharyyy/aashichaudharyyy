## 🚀 Featured Projects

### 🤖 [Twitter Bot Detection ML](https://github.com/aashichaudharyyy/twitter-bot-detection-ml)
> Machine learning model to detect Twitter bots using behavioural feature engineering.
- Trained an **SVM classifier** achieving **88% accuracy** with high precision
- Applied behavioural feature engineering on user metadata and activity patterns
- End-to-end pipeline: data collection → preprocessing → model training → evaluation
- **Stack:** Python, scikit-learn, Pandas, Jupyter Notebook

---

### 🏥 [Sepsis Prediction LSTM](https://github.com/aashichaudharyyy/Sepsis_PredictionLSTM)
> End-to-end deep learning project for early sepsis prediction using PyTorch.
- Built an LSTM-based model to predict sepsis onset from clinical time-series data
- Covers full ML pipeline: data preprocessing, feature handling, model development & evaluation
- Aimed at supporting early healthcare intervention
- **Stack:** Python, PyTorch, Pandas, Jupyter Notebook
